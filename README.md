## Prerequisites

To locally run the project you need node with the version specified in the [.nvmrc](.nvmrc) file.

## Installation
Install the dependencies with:

```
npm ci
```


## Development
Run the local development server with

```
npm run dev
```

## Usage or Deployment
To build the project run:

```
npm run build
```
